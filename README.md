# JSON Visualizer

A powerful, interactive, and beautifully designed web-based tool to visualize and explore JSON data. This tool provides multiple ways to view your data, including an interactive graph, a tree structure, and a formatted raw view.

## 🚀 Features

- **Interactive Editor**: Full-featured JSON editor powered by CodeMirror with syntax highlighting, bracket matching, and code folding.
- **Visual Graph View**: Transform complex JSON objects into an interactive node-link diagram for easier understanding of relationships.
- **Tree View**: A classic hierarchical exploration of your JSON data with collapsable nodes.
- **Raw Formatted View**: Clean, color-coded, and pretty-printed JSON view.
- **Dual Themes**: Switch between a sleek Dark mode and a clean Light mode.
- **Sample Data**: Includes pre-loaded samples (User Profile, API Response, etc.) to quickly test the visualization features.
- **Responsive Layout**: Designed with a modern, Mac-inspired UI that works beautifully across different screen sizes.

## 🛠️ Technology Stack

- **HTML5 & CSS3**: Core structure and sophisticated styling with custom CSS variables.
- **JavaScript (Vanilla)**: Core logic for data parsing and visualization.
- **CodeMirror**: Professional-grade text editor for JSON input.
- **Google Fonts**: Uses 'Outfit' for UI and 'Fira Code' for data/code.

## 📥 How to Use

1. **Clone or Download**: Simply download the `JsonVisualization.html` file.
2. **Open in Browser**: Open the file in any modern web browser (Chrome, Firefox, Safari, Edge).
3. **Input JSON**: Paste your JSON data into the editor on the left sidebar.
4. **Visualize**: Click the "Visualize" button to render the data in your selected view mode.
5. **Switch Views**: Use the toggle buttons in the header to switch between **Graph**, **Tree**, and **Raw** views.

## ⌨️ Editor Shortcuts

- `Ctrl + F`: Search within the JSON content.
- `Ctrl + G`: Find next match.
- `Ctrl + Shift + G`: Find previous match.
- `Ctrl + \`: Auto-format JSON (if implemented, otherwise use the sample buttons).
